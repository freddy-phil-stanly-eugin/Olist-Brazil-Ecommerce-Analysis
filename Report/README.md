# Report

This folder contains the final project report for the **Olist Brazil E-Commerce Analysis** case study.

## Contents

* **Olist_Brazil_Ecommerce_Analysis_Report.pdf** – Comprehensive report covering project objectives, business insights, dashboard analysis, recommendations and conclusions.

## Report Highlights

The report includes:

* Project overview and business objectives
* Dataset summary
* Executive dashboard insights
* Order analysis
* Customer & regional analysis
* Revenue & freight analysis
* Delivery & logistics analysis
* Payment behaviour analysis
* Business recommendations
* Final conclusion

The report serves as a detailed summary of the findings obtained through SQL exploration and Power BI dashboard development.
