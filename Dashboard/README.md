# Dashboard

This folder contains the Power BI dashboard developed as part of the **Olist Brazil E-Commerce Analysis** case study.

## Dashboard Overview

The dashboard was designed to transform SQL-based business insights into an interactive reporting solution. It provides a comprehensive view of customer behaviour, revenue performance, logistics efficiency and payment preferences across the Olist platform.

### Dashboard Pages

1. Executive Summary
2. Order Analysis
3. Customer & Regional Analysis
4. Revenue & Freight Analysis
5. Delivery & Logistics Analysis
6. Payment Analysis

## Key Features

* Executive KPI tracking
* Customer and regional analysis
* Revenue and freight performance monitoring
* Delivery and logistics insights
* Payment behaviour analysis
* Interactive filtering and drill-down capabilities

The dashboard was developed using Power BI and DAX measures to support business decision-making through data-driven insights.
